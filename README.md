<h1>Money Escape</h1>

<h3>Money Escape is a 2D car chasing game made with Cocoos2DX.</h3>
<h4>Game Objectives</h4>
<li>Collect money</li>
<li>Bribe the police</li>
<li>Ecape</li>
<li>Survive as much as possible</li>
